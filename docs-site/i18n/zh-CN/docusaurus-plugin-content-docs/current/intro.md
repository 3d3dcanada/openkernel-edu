# OpenKernel EDU 简介 📚

欢迎来到 OpenKernel EDU，这是世界上最易于访问的计算机科学教育平台。

## 我们的使命 🎯

我们相信，无论母语或背景如何，每个人都应该能够接触到计算机科学。这就是我们创建 **EmojiASM** 的原因——这是一种视觉汇编语言，其中每个指令都是一个表情符号 (emoji)。

## 什么是 EmojiASM？ 📥

EmojiASM 是一套简化的指令集，让学生能够学习计算机架构、内存管理和逻辑的基础知识，而不会受到复杂语法或以英语为中心的命令的阻碍。

### 主要特点：
- **100% 表情符号指令：** 零语言障碍。
- **实时可视化：** 在代码运行时观察寄存器和内存的变化。
- **多语言支持：** 文档和界面支持 6 种语言。
- **网页原生：** 无需安装，可在任何现代浏览器中运行。
