OpenKernel EDU is designed to bridge the gap between high-level logic and low-level hardware understanding.

## Why use EmojiASM for CS101?

1. **Lower Cognitive Load:** By replacing cryptic assembly mnemonics (MOV, JNZ, etc.) with familiar emojis, students can focus on the *logic* of execution.
2. **Instant Feedback:** The visual simulator shows memory and registers changing in real-time.
3. **Inclusive by Design:** Students whose first language is not English can learn the same concepts without language barriers.
